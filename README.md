# README

## QuickStart:

### Install the rrhh_etls project run 
`$ ./install.sh`

### Activate virtual env
`$ source venv/bin/activate`

### Run an ETL 
`$ rrhh_etl altas_bajas`

### To see mannual
`$ rrhh_etl -h`

## Project Structure:

### Entrypoint 
    rrhh_etl/app.py

### Config
    rrhh_etl/config.yaml
